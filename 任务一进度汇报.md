# 任务一进度汇报

## 核心功能实现
- **多文件上传**: 支持图片和视频批量上传
- **云端存储**: 集成腾讯云COS，文件自动上传至云端
- **AI关键词生成**: 集成豆包大模型，自动分析图片内容生成描述性关键词
- **素材管理**: 完整的CRUD操作（添加、读取、删除）
- **时间线视图**: 按日期组织的素材浏览界面

## 技术架构
前端请求 → Flask后端 → 腾讯云COS → 豆包大模型 → 数据库存储

## API接口清单
| 方法 | 端点 | 功能 |
|------|------|------|
| GET | `/api/materials` | 获取素材列表 |
| POST | `/api/upload` | 上传素材文件 |
| GET | `/api/materials/<id>` | 获取单个素材 |
| DELETE | `/api/materials/<id>` | 删除单个素材 |
| DELETE | `/api/materials/batch` | 批量删除素材 |
| DELETE | `/api/materials/clear` | 清空所有素材 |
| GET | `/api/timeline` | 时间线视图 |

## 数据模型
```python
class Material:
    id: str              # UUID唯一标识
    filename: str        # 原始文件名
    file_type: str       # 文件类型(image/video)
    file_path: str       # 云端存储路径
    file_size: int       # 文件大小
    upload_time: datetime # 上传时间
    ai_keywords: str     # AI生成的关键词
```

## AI生成提示词
请为该农业素材生成适合溯源故事的关键词。
                        
素材可能包含：种植、生长、管理、采摘等农业活动。

请生成15-20个描述性关键词，涵盖：
- 物品的形状、颜色、外观特征
- 农业操作和活动
- 生长过程和阶段
- 环境场景特征
- 品质管理要点（有无病虫害等）
- 人物情感（如果有的话）

用中文逗号分隔，直接返回关键词字符串。
